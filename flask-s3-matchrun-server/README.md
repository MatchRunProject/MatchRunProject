## 주요 기능
AWS Rekognition 의 객체 탐지 인공지능을 이용해 태그를 저장 할 수 있습니다. <br/>
태그로 저장된 레이블은 Naver Papago Open API를 이용해 번역되어 한글로 저장 됩니다. <br/>
